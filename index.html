Tentang Saya

Halo semua,Perkenalkan nama saya Muhammad Aditama Rachman dari Kelas XI RPL 2 SMKN 1 Tanah Grogot,nah kali ini saya akan memperkenalkan web yang pertamakali saya buat,jadi saya akan menceritakan bagaimana saya belajar membuat pertama kali web ini,tapi sebelum itu bagaimana kalau saya ceritakan dulu kenapa saya memilih jurusan RPL di SMKN 1 ini. Alasan saya memilih jurusan RPL ini adalah pada saat saya sedang bermain suatu game di komputer,pada saat bermain game itu saya berpikir,Bagaimana cara membuat game sedetail dan sebagus ini?,pada saat itu saya mencoba untuk mempelajari cara membuat game dari Youtube,nah dari situlah saya tau kalau ingin membuat game harus menggunakan tehnik coding,setelah itu,saya mencari mencari sekolah untuk belajar coding,lalu saya disarankan oleh teman smp saya untuk masuk jurusan Rpl di SMKN 1. Setelah masuk SMKN 1 Tanah Grogot dari situlah saya diajarkan tentang coding lebih mendalam,awalnya sih waktu masih belajar HTML masih gampang aja. Tapi setelah mengenal javascript dan css ketika magang disitulah saya sadar kalau membuat suatu web bukan hanya rumit tapi juga membutuhkan logika. Dalam belajar membuat web kita tidak bisa berpatokan pada 1 refrensi saja,kita harus lebih sering searching-searching di internet,bahkan kadang kita bisa belajar dari web yang berbahasa asing seperti bahasa Inggris bahkan bahasa India. setelah mempelajari tentang membuat web dari berbagai refrensi,mulailah saya membuat web yang sederhana ini,walaupun web ini sangat sederhana saya rasa ini adalah awal yang bagus buat saya untuk kedepannya mmebuat web yang lebih bagus. Itulah cerita singkat saya,maaf jika web ini terdapat kesalahan,Terima Kasih.
